# Quick Import Instructions:

## For DaVinci Resolve:
1. Create new timeline (1080x1920)
2. Import video clips
3. Go to "Titles" → "Text+"
4. Copy text from above
5. Apply animations from "Inspector" panel
6. Use color picker for exact brand colors

## For Adobe Premiere:
1. Create new sequence (1080x1920)
2. Import video clips  
3. Go to "Graphics" → "Essential Graphics"
4. Create text layers
5. Apply effects from "Effect Controls"
6. Use hex codes for brand colors

## For iMovie (Mac):
1. Create new movie project
2. Import clips in vertical orientation
3. Add titles from "Titles" browser
4. Customize text in viewer
5. Apply simple animations

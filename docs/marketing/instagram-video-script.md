# ComplyKit Instagram Promotion Video Script

## Video Concept: "Compliance Made Simple" 
**Duration**: 60 seconds (Instagram Reel)  
**Target Audience**: Developers, Startups, DevOps teams  
**Hook**: "Stop spending weeks on SOC 2 compliance..."

---

## 📱 Scene-by-Scene Breakdown

### Scene 1: The Problem (0-8 seconds)
**Visual**: Developer looking frustrated at spreadsheets and compliance docs
**Text Overlay**: "SOC 2 taking forever? 😩"
**Voiceover/Audio**: Upbeat music starts
**Screen Recording**: Show cluttered compliance spreadsheets

### Scene 2: The Solution Introduction (8-15 seconds)
**Visual**: Clean terminal with ComplyKit logo
**Text Overlay**: "Meet ComplyKit ⚡"
**Screen Recording**: 
```bash
$ brew install complykit
$ comply init
```
**Transition**: Smooth zoom into terminal

### Scene 3: Key Features - Speed (15-25 seconds)
**Visual**: Split screen - Before/After
**Text Overlay**: "From Months to Minutes"
**Screen Recording**: 
```bash
$ comply scan aws
✅ 47 checks passed
❌ 3 issues found
🔧 Auto-fix available
```
**Visual Effect**: Speed up the scanning animation

### Scene 4: Key Features - Automation (25-35 seconds)
**Visual**: Dashboard showing real-time compliance
**Text Overlay**: "Automated Evidence Collection 🤖"
**Screen Recording**: Show dashboard with live data
**Visual**: Evidence files being auto-generated

### Scene 5: Key Features - Multi-Cloud (35-45 seconds)
**Visual**: Cloud provider logos (AWS, Azure, GCP)
**Text Overlay**: "Works Everywhere 🌍"
**Screen Recording**: 
```bash
$ comply scan azure
$ comply scan gcp
$ comply scan kubernetes
```

### Scene 6: Results & CTA (45-60 seconds)
**Visual**: Success dashboard, happy developer
**Text Overlay**: "Join 1000+ Teams Already Compliant ✨"
**Screen Recording**: Clean compliance report
**CTA**: "Try ComplyKit Free → Link in Bio"

## 🎨 Visual Style Guide

### Color Palette
- **Primary**: #6366F1 (ComplyKit Blue)
- **Success**: #10B981 (Green)
- **Warning**: #F59E0B (Orange)
- **Background**: Dark terminal theme

### Typography
- **Headers**: Bold, modern sans-serif
- **Code**: Monospace font (Fira Code)
- **CTAs**: Bright contrasting colors

### Animation Style
- **Fast-paced** but readable
- **Smooth transitions** between scenes
- **Terminal animations** that feel authentic
- **Pulse effects** for important elements

---

## 🎵 Audio Elements

### Background Music
- **Style**: Upbeat, tech/startup vibe
- **BPM**: 120-140 (energetic but not overwhelming)
- **Volume**: Background level, doesn't compete with voiceover

### Sound Effects
- **Terminal typing**: Authentic keyboard clicks
- **Success sounds**: Subtle "dings" for passed checks
- **Transition whooshes**: For scene changes
- **UI clicks**: When showing dashboard interactions

### Voiceover (Optional)
- **Tone**: Friendly, confident, technical but accessible
- **Pace**: Quick but clear
- **Script**: See detailed script below

## 🎬 Production Instructions

### Pre-Production
1. **Screen Recordings Needed**:
   - Terminal commands and output
   - Dashboard navigation
   - Evidence generation
   - Report creation

2. **Assets to Prepare**:
   - ComplyKit logo variations
   - Cloud provider logos
   - Code snippets formatted nicely
   - Sample compliance reports

### Recording Setup
- **Screen Resolution**: 1080x1920 (9:16 aspect ratio)
- **Frame Rate**: 30fps minimum
- **Recording Tool**: OBS Studio or Screen Studio (Mac)
- **Terminal Theme**: Dark with high contrast

### Post-Production
- **Editing Software**: Adobe Premiere, Final Cut Pro, or DaVinci Resolve
- **Text Animation**: Kinetic typography with smooth ease-in/out
- **Color Correction**: Ensure brand colors are accurate
- **Audio Sync**: Perfect timing between visuals and audio
